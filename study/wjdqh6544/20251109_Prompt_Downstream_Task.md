### 🧩 제안하신 7가지 Downstream Task 정리

| 번호 | Task                                | 핵심 목적                          | 예시 / 설명                             |
| -- | ----------------------------------- | ------------------------------ | ----------------------------------- |
| 1  | **Anomaly Detection (이상탐지)**        | 정상 패턴에서 벗어난 데이터 포인트 탐지         | 센서 오류 탐지, 사기 거래 탐지, 시스템 이상 감지       |
| 2  | **Classification (분류)**             | 입력 데이터를 미리 정의된 클래스 중 하나로 분류    | 이미지 인식, 질병 진단, 감성 분석                |
| 3  | **Clustering (군집화)**                | 데이터의 유사성에 기반해 그룹화 (비지도 학습)     | 고객 세분화, 문서 주제 그룹화                   |
| 4  | **Forecasting (예측 / 시계열 예측)**       | 과거 데이터를 바탕으로 미래 값을 예측          | 수요 예측, 주가 예측, 날씨 예보                 |
| 5  | **Imputation (결측치 보간)**             | 누락된 데이터 값을 추정 및 복원             | 센서 데이터 결측 복원, 의료 데이터 보완             |
| 6  | **Representation Learning (표현 학습)** | 데이터의 의미 있는 저차원 표현(임베딩) 학습      | 문장 임베딩, 이미지 피처 추출                   |
| 7  | **Transfer Learning (전이 학습)**       | 한 도메인/태스크에서 학습한 지식을 다른 태스크에 전이 | 사전학습 언어모델(Phi, BERT 등) 파인튜닝, 도메인 적응 |

* ✅ **밸런스가 좋습니다** — 지도학습, 비지도학습, 자기지도학습, 전이학습을 모두 포함.
* ✅ **일관된 분류 기준** — 데이터 활용 목적(탐지/분류/예측/복원/전이 등)에 따라 명확히 구분됨.

# 시계열 분류 (Time Series Classification)

## Prompts (Powered by Claude)

### Question)
```
시계열 분류(Time Series Classification) 문제가 무엇인지, 그리고 일반적인 분류 문제와 어떻게 다른지 설명해주세요.
```

### Answer)
시계열 분류는 시간에 따라 순차적으로 관측된 데이터를 미리 정의된 범주(클래스)로 분류하는 작업입니다. 예를 들어, 심전도(ECG) 신호를 정상/비정상으로 분류하거나, 센서 데이터를 통해 사람의 행동(걷기, 뛰기, 앉기 등)을 인식하는 것이 시계열 분류 문제입니다.

**일반 분류 문제와의 차이점:**

1. **시간적 의존성**: 시계열 데이터는 데이터 포인트들 간의 순서가 중요합니다. 일반 분류에서는 특성(feature)들이 독립적이라고 가정하는 경우가 많지만, 시계열에서는 이전 시점의 값들이 현재 값에 영향을 미칩니다.

2. **가변 길이**: 시계열 데이터는 샘플마다 길이가 다를 수 있어, 고정된 크기의 입력을 가정하는 전통적 분류 모델에 직접 적용하기 어렵습니다.

3. **패턴의 형태**: 중요한 정보가 절대적인 값보다는 시간에 따른 변화의 패턴(shape)에 담겨있는 경우가 많습니다.

4. **노이즈와 왜곡**: 시계열 데이터는 측정 노이즈, 시간적 왜곡(같은 패턴이지만 속도가 다름), 위상 이동 등의 문제를 자주 겪습니다.

### Question)
```
시계열 분류에서 전통적 방법인 DTW(Dynamic Time Warping)와 Shapelet 기반 방법, 그리고 딥러닝 방법의 차이점과 각각의 장단점을 비교해주세요.
```

### Answer)
**1. DTW (Dynamic Time Warping)**

DTW는 두 시계열 간의 유사도를 측정하는 거리 기반 방법입니다. 1-NN 분류기와 결합하여 사용되며, 시간축의 비선형 정렬을 통해 속도 변화에 강건합니다.

- **장점:**
  - 구현이 간단하고 직관적
  - 시간적 왜곡(temporal distortion)에 강건
  - 해석 가능성이 높음

- **단점:**
  - 계산 복잡도가 O(n²)로 큰 데이터셋에서 느림
  - 다변량 시계열 처리가 어려움
  - 특성 학습이 없어 원시 데이터에 의존

**2. Shapelet 기반 방법**

Shapelet은 시계열의 부분 패턴(subsequence) 중 특정 클래스를 잘 구분하는 판별력 있는 패턴을 찾아 이를 특성으로 사용합니다.

- **장점:**
  - 해석 가능한 특성 제공 (어떤 패턴이 분류에 중요한지 시각화 가능)
  - 국소적 패턴을 캡처하여 전체 시계열의 일부만으로도 분류 가능
  - 노이즈에 상대적으로 강건

- **단점:**
  - Shapelet 탐색의 계산 비용이 매우 높음
  - 사전에 정의된 패턴에 의존하여 유연성이 부족
  - 복잡한 시간적 관계를 포착하기 어려움

**3. 딥러닝 방법 (CNN, RNN, Transformer 등)**

딥러닝은 다층 신경망을 통해 시계열의 계층적 표현을 자동으로 학습합니다.

- **장점:**
  - End-to-end 학습으로 수동 특성 추출 불필요
  - 복잡한 비선형 패턴과 장기 의존성 포착 가능
  - 다변량, 가변 길이 시계열 처리에 유연
  - 대규모 데이터에서 높은 성능

- **단점:**
  - 대량의 학습 데이터 필요 (시계열 분류는 종종 소규모 데이터셋)
  - 해석 가능성이 낮음 (블랙박스)
  - 하이퍼파라미터 튜닝이 복잡
  - 학습 시간과 계산 자원이 많이 필요

**실무 선택 기준:**
- 소규모 데이터셋, 해석 가능성 중요 → DTW, Shapelet
- 대규모 데이터셋, 높은 정확도 필요 → 딥러닝
- 실시간 처리, 제한된 자원 → 경량 딥러닝 모델 또는 DTW

### Question)
```
UCR/UEA 시계열 분류 벤치마크 데이터셋에 대해 설명하고, 왜 이 벤치마크가 중요한지 알려주세요.
```

### Answer)
**UCR/UEA 시계열 분류 아카이브**는 시계열 분류 연구를 위한 표준 벤치마크 데이터셋 모음입니다.

**UCR Time Series Archive:**
- 캘리포니아 대학교 리버사이드(University of California, Riverside)에서 관리
- 130개 이상의 단변량 시계열 분류 데이터셋 포함
- 다양한 도메인: 의료(ECG, EEG), 센서 데이터, 이미지 윤곽선, 모션 캡처 등
- 각 데이터셋은 미리 정의된 train/test split 제공

**UEA Time Series Archive:**
- 이스트 앵글리아 대학교(University of East Anglia)에서 관리
- 30개 이상의 다변량 시계열 분류 데이터셋 포함
- 더 복잡한 실세계 문제 포함 (여러 센서의 동시 측정 등)

**벤치마크의 중요성:**

1. **공정한 비교**: 동일한 데이터셋과 평가 프로토콜을 사용하여 서로 다른 알고리즘을 객관적으로 비교할 수 있습니다.

2. **표준화**: 연구자들이 새로운 방법을 제안할 때 이 벤치마크에서의 성능을 보고하는 것이 표준 관행이 되었습니다.

3. **다양성**: 다양한 도메인, 길이, 클래스 수, 난이도를 가진 데이터셋들이 포함되어 있어 알고리즘의 일반화 능력을 종합적으로 평가할 수 있습니다.

4. **재현성**: 공개된 데이터셋을 사용하여 연구 결과의 재현성을 보장합니다.

5. **발전 추적**: 시간에 따른 알고리즘 성능 개선을 추적하고, 현재 최고 성능(state-of-the-art)을 파악할 수 있습니다.

**활용 방법:**
연구자들은 일반적으로 UCR/UEA 아카이브의 모든 데이터셋 또는 대표적인 부분집합에 대해 평균 정확도나 순위를 보고합니다. 이를 통해 특정 데이터셋에 과적합되지 않고 일반적으로 우수한 방법을 식별할 수 있습니다.
